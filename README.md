# 🎮 Number Guessing Game

A fun, interactive command-line number guessing game built using Python. The program generates a secret random number, and the player has a limited number of attempts to guess it, receiving helpful hints along the way.

## 🚀 Features

*   **Dynamic Number Generation:** Automatically picks a random secret number between 1 and 20.
*   **Smart Hints:** Tells the user if their guess is *Too High* 📈 or *Too Low* 📉 after each attempt.
*   **Input Validation:** Robust error handling preventing crashes if a user accidentally enters a letter or an invalid symbol instead of an integer.
*   **Live Turn Tracking:** Keeps the player updated on exactly how many attempts they have remaining.

## 🛠️ How to Run the Game

### Prerequisites
Make sure you have Python installed on your computer. You can download it from [python.org](https://www.python.org/).

### Running the Script
1. Clone this repository or download the `main.py` file.
2. Open your terminal or command prompt.
3. Navigate to the folder containing the file and run:
   ```bash
   python main.py
